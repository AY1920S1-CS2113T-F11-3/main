Tip: Packaging CSV or Txt files in the JAR<\/a><\/p>\r\n

After the tutorials today I realized a few of the teams may need to include data in a txt<\/code> or csv<\/code> file for their project to work.
\r\nIf you are normally reading the file from a location, the same method doesn't work when you run the JAR.
\r\nTo include the CVS or TXT file into the JAR you can follow the example here: https://docs.gradle.org/current/userguide/java_plugin.html<\/a> (search for "project\r\n layout")**
\r\nHere is a quick tutorial to include the CSV/TXT file and be able to read it in the JAR:
